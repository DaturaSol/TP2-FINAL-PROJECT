## Layout

(see folders below)

- .devcontainer/   Shared dev environment (start here)
- .vscode/         Editor settings
- docker/          Dockerfiles + compose orchestration
- data/            SQLite3 database files (backend only)
- src/back_end/    Python - server & logic
- src/front_end/   JavaScript - user interface

## Quick start

1. Install Docker Desktop and the VS Code **Dev Containers** extension.
2. Open this folder in VS Code → **Reopen in Container** (first build takes a
   few minutes).
3. Backend devs: see `src/back_end/README.md`.
   Frontend devs: see `src/front_end/README.md`.

The backend serves on **port 8080**; the frontend reaches it at
`http://localhost:8080`.
